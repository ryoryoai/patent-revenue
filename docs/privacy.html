Patent Value Analyzer

プライバシーポリシー

Patent Value Analyzer（以下「本サービス」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

1. 収集する情報

本サービスは、診断結果のメール送信機能をご利用いただく際に、以下の情報を収集します。

  • メールアドレス（必須）: 診断結果の送付先として使用します。
  • 氏名（任意）: メール送信時の宛名として使用します。

なお、診断機能の利用のみの場合、個人を特定する情報は収集しません。診断入力（特許番号・キーワード）の一時キャッシュは同一入力の再計算回避を目的としています。

2. 利用目的

収集した情報は、以下の目的にのみ使用します。

  • 特許の簡易診断結果の提供
  • 診断レポートのメール送信
  • 詳細登録への案内
  • 特許の市場動向調査・活用可能性の分析（当社内部での利用に限り、結果を外部に公開することはありません）
  • サービス改善のための統計分析（個人を特定しない形で実施）

3. データの保存

入力いただいた特許情報および連絡先は、当社のデータベースに保存します。これは診断結果の送付、今後のご案内、および当社内部での特許分析に利用するためです。保存期間はサービスの利用目的が達成されるまでとし、ユーザーからの削除依頼には本ポリシーに定める手続きに従い対応します。

4. 第三者提供

収集した個人情報を、法令に基づく場合を除き、ユーザーの同意なく第三者に提供することはありません。

メール送信には外部サービス（Resend）を利用しますが、メール送信の目的以外に情報が使用されることはありません。

5. ユーザーの権利

ユーザーは、自己の個人情報について以下の権利を有します。

  • 開示・訂正・削除の請求
  • 利用停止の請求

これらのご請求は、下記お問い合わせ先までご連絡ください。

6. Cookie・アクセス解析

本サービスは、不正アクセス対策のためにセッション管理用Cookieを使用します。また、Cloudflare Turnstileによるbot検知を行う場合があります。

7. 改定

本ポリシーは、法令の変更やサービス内容の変更に応じて改定することがあります。重要な変更がある場合は、本ページにて通知します。

8. お問い合わせ

プライバシーに関するお問い合わせは、以下までご連絡ください。

メール: privacy@iprich.jp
